Acerca del chat

Esta es una pagina prototipo, debe eliminarse en un futuro

Ingrese el ID de una sala, si la sala no existe, esta se creará, si ya existe, te unirás a ella

Por el momento no se hacen validaciones sobre el nickname, pueden haber varios usuarios con el mismo nickname y no representa un problema

Es posible acceder a diferentes salas de chat en usando varias pestañas (no es necesario usar el modo incognito)

La comunicacion se realiza mediante JSON. Cada mensaje posee el ID de la sala, el nickname del usuario y el mensaje enviado. Dada esta modularidad, es posible modelar la visa del mensaje modificando el codigo HTML de la pagina del lado del cliente mediante JavaScript
